# Student Data Collection – Personal Dashboard Application

## About the Project

The **Personal Dashboard Application** is a simple web-based application that generates personalized information based on a user's name and age.

The application provides:

- Personalized greetings
- Age calculations
- User status classification
- Motivational reminders
- Daily inspirational quotes

Future enhancements will include generating motivational messages based on the day of the week and integrating real-time world news updates.
## Features

- Personalized user greeting
- Age calculation and conversion
- User status classification (Minor/Adult)
- Motivational reminders
- Daily inspirational quote section
- Responsive and modern user interface
- Dynamic content generation using JavaScript
## Built With

- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

### Installation

1. Clone the repository:

```bash
git clone https://github.com/jblue254/calorie_counter.git
2. Navigate to the project directory:

```bash
cd calorie_counter
3. Open the project in your preferred code editor.

4. Launch `index.html` in your browser.
## Usage

1. Enter your name.
2. Enter your age.
3. Click the **Generate Dashboard** button.
4. View your personalized dashboard information.
